## UI Components

### 1. **Scene Image** (Top Section)
- Full-width background image
- Generated by Illustrator agent
- Shows character in the scenario
- Scene number badge (top-left)
- Coin earned badge (top-right, if completed)

### 2. **Scenario Box** (Middle Section)
- Rounded box with gradient background
- 1-2 sentences describing situation
- Includes character name
- Easy-to-read font

### 3. **Question** (Center)
- Large, bold text
- "What should [Character] do?"
- "How should [Character] respond?"

### 4. **Choice Buttons** (Two Options)
- **Option A** (Green) - Usually correct
- **Option B** (Red) - Usually wrong
- Large, tappable buttons
- Clear text with emojis

### 5. **Feedback Display**
- Replaces options after selection
- Shows emoji (ðŸŒŸ for correct, ðŸ’­ for wrong)
- Displays feedback message
- "Try Again" button for wrong answers

### 6. **Navigation Bar** (Bottom)
- **Previous Button** (â—€)
  - Disabled on page 1
  - Goes back one scene
  - Preserves completed state
  
- **Progress Dots** (â—â—â—â—‹â—‹â—‹â—‹â—‹)
  - 8 dots for 8 scenes
  - Current page: Large purple dot
  - Completed: Green dot
  - Not started: Gray dot
  
- **Next Button** (â–¶)
  - Disabled until scene completed
  - Goes forward one scene
  - Disabled on page 8

### 7. **Coin Counter** (Top-Right)
- Shows coins earned / total (e.g., "3/8")
- Updates in real-time
- Gold coin icon

## Interaction States

### State 1: Fresh Scene (Not Answered)
```
- Image displayed
- Scenario visible
- Question visible
- Both option buttons enabled
- Next button disabled
```

### State 2: Wrong Answer Selected
```
- Feedback shown (orange box)
- "Try Again" button
- Options hidden
- Next button still disabled
```

### State 3: Correct Answer Selected
```
- Feedback shown (green box)
- +1 coin animation
- Progress dot turns green
- Auto-advance after 2 seconds
- OR Next button enabled
```

### State 4: Scene Completed
```
- Image displayed
- Scenario visible
- Question visible
- Options disabled (grayed out)
- Next button enabled
- Can go back to previous scenes
```

## Visual Design

### Color Scheme
- **Background:** Gradient purple/pink/blue
- **Book Page:** White with amber border
- **Header:** Purple-to-pink gradient
- **Correct Option:** Green tones
- **Wrong Option:** Red tones
- **Feedback (Correct):** Green background
- **Feedback (Wrong):** Orange background
- **Navigation:** Purple buttons

### Typography
- **Title:** Bold, 2xl
- **Scenario:** Medium, lg
- **Question:** Bold, 2xl, purple
- **Options:** Semibold, lg
- **Feedback:** Semibold, xl

### Spacing
- Generous padding (p-6, p-8)
- Clear separation between sections
- Rounded corners (rounded-2xl, rounded-3xl)
- Shadow effects for depth

## Responsive Design

### Desktop (Large Screens)
- Max width: 4xl (896px)
- Two-column option layout
- Larger text and buttons

### Tablet (Medium Screens)
- Max width: 3xl (768px)
- Two-column option layout
- Medium text and buttons

### Mobile (Small Screens)
- Full width with padding
- Single-column option layout
- Stacked buttons
- Smaller navigation arrows

## Accessibility Features

1. **Clear Visual Hierarchy**
   - Scene number always visible
   - Question stands out
   - Options clearly separated

2. **Color + Text**
   - Not relying on color alone
   - Emojis for visual cues
   - Clear labels (A, B)

3. **Large Touch Targets**
   - Buttons are big and tappable
   - Generous padding
   - Clear hover states

4. **Progress Indication**
   - Page counter (1 of 8)
   - Progress dots
   - Coin counter

5. **Feedback**
   - Immediate response to clicks
   - Clear success/error states
   - Encouraging messages

## Animation & Transitions

1. **Page Transitions**
   - Smooth fade between scenes
   - Slide animation (optional)

2. **Button Interactions**
   - Hover: Scale up (scale-105)
   - Click: Slight press effect
   - Disabled: Opacity 50%

3. **Coin Earned**
   - Pop-up animation
   - Sparkle effect
   - Counter increment

4. **Auto-Advance**
   - 2-second delay after correct answer
   - Smooth transition to next page

## Component Props

```typescript
interface QuestBookProps {
  questTitle: string           // "Mila's Sharing Adventure"
  characterName: string         // "Mila the Star Bunny"
  scenes: QuestScene[]         // Array of 8 scenes
  onQuestComplete: (coins: number) => void  // Callback when done
}

interface QuestScene {
  scene_number: number         // 1-8
  scenario: string             // Situation description
  question: string             // What should X do?
  option_a: Option            // Correct choice
  option_b: Option            // Wrong choice
  image_uri: string           // Scene illustration URL
}

interface Option {
  text: string                // Button text
  is_correct: boolean         // true/false
  feedback: string            // Response message
}
```
